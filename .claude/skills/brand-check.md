# Brand Check Skill

## Description
コンテンツ・コピー・ビジュアルがHoneyTrapのブランドガイドラインに適合しているか総合チェックする。

## Trigger
`/brand-check` [対象ファイルまたはテキスト]

## Checklist

### ブランドトーン
- [ ] 「バンドの公式サイト」として自然か（漫画紹介サイトになっていないか）
- [ ] 「漫画発」であることは匂わせつつ、バンドとしてのリアリティがあるか
- [ ] トーン: Edgy but Approachable か

### DO（推奨）
- [ ] メンバーを実在のバンドメンバーとして語っている
- [ ] chao!を「ビジュアル担当アーティスト」として紹介している
- [ ] 「漫画から飛び出してきた」演出がある（適度に）
- [ ] 90年代〜Y2Kの空気感がある

### DON'T（禁止）
- [ ] 「漫画のキャラクター」「フィクションの〜」と直接言及していない
- [ ] 子供向け・ファンシーすぎるトーンになっていない
- [ ] 既存バンドの劣化コピーになっていない
- [ ] 過度な説明（世界観は体験させる、説明しない）がない

### ビジュアル
- [ ] chao!のアートスタイル（カラフル・ポップ・レトロ）と整合する
- [ ] メンバーカラーが正しく使用されている
- [ ] ダークUI原則に従っている
