# HoneyTrap IP戦略

## IP展開ロードマップ

### Phase 1: 認知獲得期（0〜6ヶ月）
- **目標**: コアファン1万人獲得
- 漫画連載 + SNS開設（メンバーが"本人"として投稿）
- オリジナル楽曲1〜2曲リリース + MV公開
- オフィシャルサイト公開
- **KPI**: SNS 1万フォロワー / 楽曲10万再生 / サイト月間5,000UU

### Phase 2: IP確立期（6〜18ヶ月）
- **目標**: 音楽IPとしての実体化
- フルアニメーションMV / 1st EP リリース
- グッズ第1弾 / ファンクラブ開始
- 漫画エピソードと楽曲リリースの同期
- **KPI**: 楽曲100万再生 / グッズ月商100万 / ファンクラブ2,000人

### Phase 3: 体験拡張期（18〜36ヶ月）
- **目標**: リアル体験の創出
- バーチャルライブ / コラボカフェ / 2nd アルバム
- 他アーティストとのコラボ
- アニメ化企画始動
- **KPI**: ライブ動員5万人 / 楽曲1,000万再生

### Phase 4: フランチャイズ化期（36ヶ月〜）
- TVアニメ放送 / 大型ライブイベント / 海外展開
- **KPI**: IP年間売上1億円 / 海外ファン比率20%

## サイトの役割

### やること
- バンドの"公式ホーム"（世界観の起点）
- 音楽ハブ（ストリーミングリンク集約）
- ニュース配信 / 世界観への入口 / EC導線 / メディア向け窓口

### やらないこと
- 漫画ビューア・漫画紹介サイト化
- 「架空のバンドです」という説明
- メンバーの"中の人"情報の掲載
- 過度な広告表示

## ファン導線
1. **初接触** → SNS/漫画/配信プラットフォームからサイトへ
2. **興味** → メンバー把握・楽曲試聴・世界観理解
3. **体験** → MVフル視聴・SNSフォロー・プレイリスト追加
4. **ファン化** → メンバーシップ登録・Discord参加
5. **エンゲージメント** → ライブ参加・ファンアート投稿
6. **課金** → 音楽DL・グッズ・ファンクラブ有料枠・ライブチケット

## 競合との差別化
- **vs ぼざろ**: 最初から「実在前提」。声優/演者を前面に出さず没入感維持
- **vs Gorillaz**: 漫画という「成長ストーリー」をリアルタイムで追体験できる
- **vs YOASOBI**: キャラクター自体がIPの主体。グッズ/ライブの展開余地が広い
- **vs 初音ミク**: 明確な物語とパーソナリティ。「推す」「見守る」関係性
