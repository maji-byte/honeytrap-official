# HoneyTrap ブランドコンセプト

## コアメッセージ

> 「漫画のコマから鳴り出した音楽が、画面を突き破って現実を揺らす。HoneyTrapは"物語の中のバンド"じゃない。今この瞬間、あなたのイヤホンの中で鳴っている、本物のロックバンドだ。」

## 世界観定義

HoneyTrapの世界は、**フィクションとリアルの「継ぎ目」が見えない場所**にある。

| レイヤー | 扱い | 具体例 |
|---|---|---|
| 漫画内の物語 | 「裏側」「ルーツ」として参照 | バンド結成の経緯、メンバーの関係性 |
| 音楽・MV | 完全に「リアル」として提示 | Spotify配信、YouTube MV |
| アーティストサイト | リアルなバンドのオフィシャルサイト | ニュース、ディスコグラフィ |
| 漫画との接点 | さりげなくリンクさせる | 「原作漫画」ではなく「連載中のドキュメント」的な紹介 |

## トンマナ原則

### 5つのDO
1. **ラウドに、ビビッドに** — 色もレイアウトも音量を上げる
2. **レトロとモダンを衝突させる** — VHSテクスチャ＋現代グリッド
3. **バンドの体温を感じさせる** — 情報サイトではなく「居場所」
4. **紙とデジタルの境界を遊ぶ** — 漫画のコマ割りをUI構造に溶け込ませる
5. **ファッションとして成立させる** — スクショがInstagramに載せたくなるデザイン

### 5つのDON'T
1. **「可愛いだけ」にしない** — ロックの骨格を持たせる
2. **説明的にならない** — 体験で世界観を理解させる
3. **白背景に整然と並べない** — 音楽サイトのエネルギーを殺さない
4. **既存バンドサイトをなぞらない** — HoneyTrap固有の文法をつくる
5. **動きを足しすぎない** — 一つ一つの動きに「音」を感じさせる

## ムードボード言語化

- **色**: 深いマゼンタ、ネオンティール、ホットピンク、ネオンイエロー。夜のネオンが濡れたアスファルトに反射する光と影
- **空気感**: ライブハウスの地下階段、ドアの向こうからベースの低音が漏れる「始まる直前」の空気
- **温度**: 28℃、湿度高め。汗ばむような熱量。夏祭りやフェスの心地よい暑さ
- **テクスチャ**: ブラウン管の走査線、写ルンですの粒子、ステッカーだらけのギターケース
- **音**: サビ前の一瞬の静寂→全員が一斉に鳴る爆発力。ページスクロールが楽曲のダイナミクスと同期する体験
